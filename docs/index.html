    Basic grids

    Standard, fluid responsive layouts.

    Currently only 3, 4, 6, 8, and 12 columns are available, but more will be added in the future. All column classes are in the format of .col-*. (You probably have known this, but make sure you file them under .col too.)

    Grid offset

    No more empty .col-*'s with &nbsp;. In the case you need some space (oh don't we all), just add .offset-* and you're good to go.

    .col-2 .offset-10
    .col-2 .offset-8
    .col-2 .offset-6
    .col-2 .offset-4
    .col-2 .offset-2
    .col-2

    Pushing a column

    Pushing a column switches the position of the grid without sacrificing HTML hierarchy.

    With push
    .col-8 .push
    .col-4
    Without push
    .col-8
    .col-4
